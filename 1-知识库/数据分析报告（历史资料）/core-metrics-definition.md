# 核心指标定义

## 投放层指标
### CPM
- 含义：千次展示成本
- 用途：判断流量购买成本

### CTR
- 含义：点击率
- 用途：判断素材和广告吸引力

### CPC
- 含义：单次点击成本
- 用途：判断点击获取成本

## 到站层指标
### Landing Page View
- 含义：有效到达落地页的访问
- 用途：判断广告点击到页面加载的实际承接情况

### 停留 / 跳出
- 含义：用户进入页面后的停留和离开情况
- 用途：判断页面承接和内容吸引力

## 转化层指标
### 询盘 / 留资 / 线索
- 含义：用户产生有效业务动作
- 用途：判断最终转化质量

## 当前使用原则
- 不能只看 CTR
- 需要按 投放层 → 到站层 → 转化层 顺序判断问题
- 高点击不等于高质量流量
