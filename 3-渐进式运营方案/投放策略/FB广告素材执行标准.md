# FB 广告素材执行标准

**日期**: 2026-04-02  
**适用范围**: BM1测试组、BM2放量组

---

## 一、BM1 测试组标准(周晓如)

### 投放标准

**什么时候投放**:
- 新素材制作完成后,立即上线测试
- 每天上午10点前上线当日新素材
- 每个素材独立建广告组,不混在一起

**初始预算**:
- 单个素材: $30-50/天
- 单个投手: $100-150/天
- BM1组总预算: **不超过 $300/天**

**定向设置**:
- 国家: 美国
- 性别: 女性
- 年龄: 25-34岁(90后) 或 18-24岁(Z世代)
- 兴趣: 自动定向或宽泛兴趣

---

### 暂停标准

**立即暂停**(跑满 $20 或 100次展示后判断):
- CTR <1.0% → 素材吸引力太差
- CPC >$1.0 → 点击成本过高
- 花费 $50 但 HVU=0 → 完全无效

**24小时后暂停**(跑满 $50 后判断):
- CTR <1.5% → 低于最低标准
- CPC >$0.8 → 学习期成本过高
- HVU单价 >$25 → 转化成本过高

**48小时后暂停**(跑满 $100 后判断):
- CTR <2.0% → 未达到及格线
- HVU单价 >$20 → 转化效率低
- 连续2天无HVU产出

---

### 传递给 BM2 的标准

**符合以下条件,立即通知黄梓铭放量**:
- ✅ CTR ≥3.0%
- ✅ CPC <$0.6
- ✅ HVU单价 <$15
- ✅ 至少产出 2个HVU(证明转化稳定)

**传递方式**:
- 每天下午6点前,周晓如在群里同步高效素材
- 提供: 素材文件、广告组ID、关键数据(CTR/CPC/HVU单价)
- 黄梓铭确认后,当晚或次日上线到BM2

---

## 二、BM2 放量组标准(黄梓铭)

### 投放标准

**什么时候投放**:
- 接收到周晓如验证过的高效素材后,立即上线
- 或者复制自己已验证的高转化素材,制作变体

**初始预算**:
- 单个素材: $100-200/天
- 单个投手: $300-500/天
- BM2组总预算: **不超过 $700/天**

**定向设置**:
- 与BM1测试时的定向保持一致
- 或者扩大年龄范围(18-45岁)测试更大市场

---

### 放量标准

**什么时候加大预算**:

**第1天**(学习期):
- 初始预算: $100/天
- 如果前4小时CTR >2.5% → 加到 $150/天
- 如果前8小时产出1个HVU → 加到 $200/天

**第2-3天**(学习期结束):
- 如果CTR稳定 >3.0% + HVU单价 <$15 → 加到 $300/天
- 如果CTR >4.0% + HVU单价 <$12 → 加到 $500/天

**第4天+**(稳定期):
- 如果持续稳定产出 → 保持预算或继续加大
- 每次加预算幅度: 20-50%,不要翻倍(避免触发重新学习)

---

### 暂停标准

**立即暂停**:
- 广告组被拒审或收到警告
- 账号状态异常(受限、警告)

**24小时后暂停**:
- CTR 从 >3% 掉到 <2% → 素材疲劳
- HVU单价 从 <$15 涨到 >$20 → 转化变差
- 连续24小时无HVU产出

**降预算观察**(不直接暂停):
- CTR 从 >3% 掉到 2-2.5% → 降预算50%观察
- HVU单价 从 <$15 涨到 $15-18 → 降预算30%观察

---

### 优化动作

**素材疲劳处理**:
- 制作素材变体(换文案、换封面、换钩子)
- 复制广告组,更换定向人群
- 暂停原素材3-5天后重新上线

**定向优化**:
- 排除已转化用户(避免重复触达)
- 扩大或缩小年龄范围
- 测试不同兴趣标签

## 三、预算控制总览

### 每日预算上限
| 组别 | 单素材预算 | 单投手预算 | 组总预算 |
|------|----------|----------|---------|
| BM1测试组 | $30-50 | $100-150 | **$300** |
| BM2放量组 | $100-200 | $300-500 | **$700** |
| **全公司** | - | - | **$1000** |

### 预算分配建议
- BM1: 30% 预算用于测试
- BM2: 70% 预算用于放量
- 如果BM1发现大量高效素材,可临时调整比例

### 超预算处理
- 单个广告组超预算50% → 立即检查,手动降预算
- 单个投手超预算 → 组长介入,暂停部分广告组
- 全公司超预算 → 黄楠楠立即通知你,暂停新测试

## 四、域名和落地页配置

### 当前情况
- 只有1个独立站
- 需要准备2个域名(BM1和BM2隔离)

### 解决方案

**方案A: 主域名 + 子域名(推荐)**
```
主域名: creviatech.com (BM2放量用)
子域名: ads.creviatech.com (BM1测试用)
```

**优点**:
- 只需要1个独立站,内容完全一致
- 技术实现简单,曾凡立配置DNS即可
- 两个域名指向同一个网站

**配置步骤**:
1. 曾凡立在DNS添加A记录: `ads.creviatech.com` → 服务器IP
2. 服务器配置支持两个域名访问同一个网站
3. Facebook像素分别配置:
   - BM1绑定 `ads.creviatech.com`
   - BM2绑定 `creviatech.com`

---

**方案B: 完全独立的两个域名**
```
域名1: creviatech.com (BM2放量用)
域名2: craveai-shop.com (BM1测试用,新注册)
```

**优点**:
- 完全隔离,一个被封不影响另一个
- 更安全

**缺点**:
- 需要注册新域名($10-20/年)
- 需要配置两个域名都指向同一个网站
- 需要配置两套SSL证书

**配置步骤**:
1. 注册新域名 `craveai-shop.com`
2. 曾凡立配置DNS指向服务器IP
3. 配置SSL证书
4. 服务器支持两个域名访问
5. Facebook像素分别配置

---

**推荐**: 先用方案A(主域名+子域名),快速上线。如果后续域名被封风险高,再启用方案B。

### 技术团队任务(曾凡立)
- [ ] 配置子域名 `ads.creviatech.com`
- [ ] 确保两个域名都能访问独立站
- [ ] 为BM1创建独立的Facebook像素
- [ ] 为BM2创建独立的Facebook像素
- [ ] 测试两个域名的追踪是否正常

## 七、常见问题

### Q1: 素材CTR很高但没有HVU怎么办?
**A**: 说明素材吸引了错误人群,或落地页承接有问题
- 检查定向是否准确
- 检查落地页内容与素材是否匹配
- 暂停该素材,调整后重新测试

### Q2: BM1测试的素材,BM2放量后效果变差?
**A**: 可能是定向人群饱和或素材疲劳
- 扩大定向范围(年龄、兴趣)
- 制作素材变体
- 降低预算,避免过度曝光

### Q3: 两个BM可以同时跑同一条素材吗?
**A**: 可以,但要错开时间
- BM1测试完,暂停后,BM2再上线
- 避免两个BM同时跑,会互相竞争

### Q4: 预算用完了但还想测试新素材怎么办?
**A**: 暂停表现差的素材,腾出预算
- 优先暂停CTR <2%的素材
- 或者等第二天预算刷新

---

**核心原则**: 
- BM1小预算快速试错,找到好素材
- BM2大预算稳定放量,收割HVU
- 所有决策回到"HVU单价"这个指标
