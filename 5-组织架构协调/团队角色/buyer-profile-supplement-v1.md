# Crave AI 投放投手资料补充表 v1

## 一、这份补充表的用途
这份表不是做最终定岗，而是先把当前已知的历史投放信息、素材类型、常见问题和后续适配方向沉淀下来，方便你后面做三件事：
- 判断谁更适合承担什么测试任务
- 判断账号风险和素材放量风险会先在哪个投手动作上放大
- 后续根据员工情况再做更细的安排

当前原则：
- 先做能力识别，不做重组织调整
- 先按历史动作和素材类型分层
- 先识别风险，再安排更重的任务

## 二、投手资料补充表

### 1. 黄梓铭
- **当前已知角色**：最熟悉现有投放操作流程，是当前 Facebook 投放经验最完整的人
- **历史投放覆盖**：历史工作表中投放记录最多，已覆盖剧场详情路径、产品详情路径、不同题材视频投放、产品型素材投放
- **更熟的路径类型**：
  - 视频跳转到剧场详情
  - 视频跳转到产品详情
  - 产品单品详情页承接
- **当前暴露出的常见问题**：
  - 多个广告组出现优质访问数量/单价偏低
  - 多个产品详情页路径存在优质访问占比偏低
  - 部分剧场路径 CTR 不一定差，但高质量访问承接弱
- **当前可见优势**：
  - 对现有投放流程熟
  - 能承担对照测试、定向测试、放量执行
  - 适合承接“已验证方向的小幅放量”
- **当前风险点**：
  - 在素材起量后容易进入“继续操作放大”阶段，此时最容易把账号风险和判断失真一起放大
  - 容易把问题理解为投放问题，但很多历史样本已经显示承接侧也在拖累结果
- **当前更适合承担的任务**：
  - Facebook 主投放位
  - 已验证素材的小步放量
  - 国家/语言/设备对照测试
  - 账号矩阵承压下的执行测试
- **当前不建议承担过重的部分**：
  - 在账号池不稳时承担过激进的放量动作
  - 在承接问题没拆清时独立决定大规模复制
- **你后续管理时对他的关键抓手**：
  - 盯高质量用户成本，不只盯 CTR/CPC
  - 盯同类素材放量边界
  - 盯账号矩阵是否足够支撑连续投放

### 2. 黄楠楠
- **当前已知角色**：最懂独立站设计，目前也参与投放
- **历史投放覆盖**：有产品视频跳转到产品详情、剧场详情、图片跳转到详情等投放记录
- **更熟的路径类型**：
  - 产品详情页承接
  - 剧场详情页路径
  - 能把页面理解带入投放判断
- **当前可见优势**：
  - 对独立站设计理解更深
  - 更适合识别“素材承诺”和“页面承接”是否一致
  - 适合做落地页与素材匹配判断
- **当前风险点**：
  - 如果后面让她同时承担太多纯投放执行，设计/承接判断优势可能被淹没
  - 更适合做带有页面反馈性质的投放，不适合只按粗放跑量来用
- **当前更适合承担的任务**：
  - 页面与素材一致性判断
  - 关键承接页反馈
  - 页面跳转逻辑与用户停留质量判断
  - 产品页/剧场页承接问题排查
- **你后续管理时对她的关键抓手**：
  - 让她给“为什么用户没有继续走”提供页面视角判断
  - 把她更多放在承接优化反馈位，而不是单纯消耗预算位

### 3. 周晓如
- **当前已知角色**：内容输出能力强，逻辑性强，数据能力也好，当前也需要实际投放，但投放剂量可以先小一些
- **历史投放覆盖**：有图片跳转首页、视频跳转剧场详情、文章详情、英文剧本详情等方向
- **更熟的路径类型**：
  - 情绪钩子型内容
  - 人设驱动型剧情内容
  - 文章/内容型详情页
- **当前可见优势**：
  - 很适合做“内容建立兴趣”这一步
  - 擅长写出前 3 秒钩子、角色设定、共鸣文案
  - 对内容逻辑和情绪组织更有优势
  - 数据能力较强，适合承担小组长位上的判断与复盘
- **当前风险点**：
  - 如果投放剂量一下拉太高，容易在操作层和判断层被短期数据牵着走
  - 如果只让她做内容、不让她跑投放全流程，会限制她作为小组长的成长和判断完整度
- **当前更适合承担的任务**：
  - 新内容方向测试
  - 情绪钩子/剧情开头/文案角度验证
  - 英文内容与文章详情路径测试
  - 小剂量实际投放，补全从内容到投放的数据判断
  - 以小组长视角参与复盘、拆解和节奏判断
- **当前不建议承担过重的部分**：
  - 独立负责高风险账号下的大幅放量动作
  - 复杂账户节奏与大规模操作测试
- **你后续管理时对她的关键抓手**：
  - 重点看她带来的用户质量，而不是只看能否起量
  - 让她在小剂量投放里跑完整链路，形成可带人的判断标准
  - 把她放在小组长位，兼顾内容判断、数据复盘和节奏管理

### 4. 陈洪骏
- **当前已知角色**：文字内容能力强，适合承接内容表达与题材包装，当前也需要实际投放，但可以先控制剂量
- **历史投放覆盖**：已覆盖剧场详情、产品详情、文章详情等，且背景描述明显更细，说明其内容包装能力更强
- **更熟的路径类型**：
  - 强人设剧情内容
  - 产品卖点展示内容
  - 文章型内容入口
- **当前可见优势**：
  - 能把题材、情绪、角色和内容目的讲清楚
  - 适合做“从内容兴趣到继续浏览”的承接前段
  - 在内容命题和包装上比较有价值
- **当前风险点**：
  - 如果没有统一判断口径，容易停在“内容表达好”但未必能形成高质量用户
  - 需要和页面承接、推荐路径绑定看，不然容易只优化前端吸引力
- **当前更适合承担的任务**：
  - 新题材内容包装
  - 人设/情绪型内容方向探索
  - 图文与视频脚本的表达优化
  - 文章详情页与剧情详情页前段吸引力优化
  - 小剂量实际投放，验证内容包装能否转成有效承接
- **你后续管理时对他的关键抓手**：
  - 让他输出的内容直接对应转化路径，不只服务点击
  - 盯他产出的内容是否把用户顺利带进合理跳转页
  - 让他通过小剂量投放补齐内容到结果的闭环判断

### 5. 陆宗林
- **当前已知角色**：主要负责寻找渠道，顺带做投放
- **历史投放覆盖**：JuicyAds、TrafficStars、ExoClick 等渠道测试较多
- **更熟的路径类型**：
  - 非 Facebook 渠道测试
  - 成人站流量验证
  - 渠道差异化流量尝试
- **当前可见优势**：
  - 更适合做渠道探索
  - 有利于做平台差异对照
  - 适合做流量来源扩展和规则理解
- **当前风险点**：
  - 这类渠道的数据判断口径和 Facebook 不一样，容易误判
  - 如果没有单独规则，CTR、点击到访问率等指标容易被看错
- **当前更适合承担的任务**：
  - 新渠道探索
  - 不同平台规则整理
  - 非 Facebook 渠道验证
  - 渠道侧风险前置排查
- **你后续管理时对他的关键抓手**：
  - 让他先交付渠道有效性判断，不先要求复杂转化结论
  - 把他放在“渠道发现”和“平台规则适配”位更合理

### 6. 潘略志
- **当前已知角色**：剪辑背景，当前产能约半人
- **历史工作表中暂未看到明确投放历史沉淀**
- **当前可见优势**：
  - 更可能在素材制作、视频处理、剪辑支持上发挥作用
- **当前风险点**：
  - 当前产能有限，不适合承担连续高强度投放任务
  - 若直接放到关键投放位，容易因节奏不稳定影响整体执行
- **当前更适合承担的任务**：
  - 素材编辑支持
  - 视频版本拆分
  - 已验证方向的素材变体支持
- **你后续管理时对他的关键抓手**：
  - 把他当“素材产能补充位”而不是核心投放位

## 三、当前阶段的渐进式安排建议

### 第一层：先稳定主执行
- **黄梓铭**：主投放执行与小步放量
- **陆宗林**：渠道探索与平台规则补充

### 第二层：把内容优势和轻投放一起跑出来
- **周晓如**：内容兴趣建立 + 小剂量投放验证 + 小组长复盘位
- **陈洪骏**：题材包装、文案与内容承接前段 + 小剂量投放验证

### 第三层：把承接判断接上
- **黄楠楠**：页面承接、素材与页面匹配、关键路径反馈

### 第四层：做素材产能补充
- **潘略志**：剪辑与素材变体支持

## 四、你现在最该盯的不是“谁强谁弱”，而是四件事
1. 谁更适合负责**内容建立兴趣**
2. 谁更适合负责**执行与小步放量**
3. 谁能帮助你更快识别**页面承接问题和账号风险**
4. 谁在跑完整个运营流程后，已经具备**带人和复盘的能力**

这比现在就做复杂人员协同更重要。
